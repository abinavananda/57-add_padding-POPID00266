Padding

We use the padding property to add padding to all sides of the container. padding: 20px; adds 20 pixels of padding to the top, right, bottom, and left sides of the container. You can adjust the value of padding to suit your design needs. You can specify different values for individual sides using padding-top, padding-right, padding-bottom, and padding-left properties if you need different amounts of padding on each side.

.container { padding: 20px 40px 80px 40px; }
